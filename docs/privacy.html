Privacy Policy

Last updated: 5 June 2026

Mirror is a local, open-source Claude Code plugin. It is built so your data never leaves your machine. This policy explains what that means in practice.

What Mirror does with your data

  • Mirror reads the transcript files Claude Code already writes under ~/.claude/projects on your own computer, and renders them as a local web page.
  • The Mirror server binds only to 127.0.0.1 (localhost). The page is reachable only from your own machine.
  • Mirror builds a local search index (SQLite) at ~/.mirror/index.db, derived from those transcripts. It stays on your machine and you can delete it at any time; it rebuilds from the transcripts.

What Mirror does not do

  • It does not call any LLM or external API.
  • It does not send your transcripts, prompts, code, or any usage data anywhere.
  • It contains no analytics, telemetry, or tracking. The rendering libraries it uses (marked, highlight.js, mermaid) are bundled with the plugin; nothing is fetched from a CDN at runtime.
  • It does not collect personal information.

Network activity

The only network activity the plugin performs is the local server listening on 127.0.0.1. Installing or updating Mirror downloads the plugin from GitHub, which is a standard git operation you initiate, not something the plugin does on its own at runtime.

This documentation site

This page and the Mirror landing page are static pages hosted on GitHub Pages. Visiting them is subject to GitHub's privacy practices (for example, standard web server logs). The Mirror plugin itself sends nothing to these sites.

Changes

If this policy changes, the updated version will be published at this URL with a new "last updated" date.
